The roster

Ten principal engineers, one muscle memory.

One connected panel, one cell per persona. Each carries the same shape: a domain, a playbook, when-to-equip conditions, the skills it brings, and a verification bar. Learn one and the next nine read the same. Backend stays lit as the default persona.

Equip one

Match a task, equip a persona.

Point the engine at a task or a work bead and it returns the best-fit persona, its score, and the terms that fired. Equip it and the persona overlays the session for that task.
